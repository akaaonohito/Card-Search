# ポケモンカード サジェスト検索Webアプリ

ポケモンカードの名前や収録パックなどの情報を高速に検索し、コピーしやすいフォーマットで出力するためのクライアントサイドWebアプリケーションです。

## 主な機能

- **サジェスト検索**: カード名、ひらがな、ローマ字、収録パックコード（例: `sv7a`）、コレクター番号などでカードを検索できます。
- **キーボード操作対応**: 矢印キー（↑ / ↓）での候補選択、Enterキーでの確定に対応しています。
- **ワンクリックコピー**: 検索して選択したカード情報をボタン一つでクリップボードにコピーできます。
  - 出力フォーマット例: `ハギギシリ sv7a 068/064 AR`

## 使い方

1. `index.html` をブラウザで開きます。
   - ※ `cards.json` を非同期で読み込むため、ローカル環境の場合は簡易的なローカルサーバー（VSCodeのLive Server拡張機能や、Pythonの `python -m http.server` など）を立ち上げて開くことを推奨します。
2. 検索窓にカード名やパックのコードを入力します。
3. 表示されたサジェスト候補から目的のカードを選択（クリックまたは矢印キー＋Enter）します。
4. 「コピー」ボタンを押してクリップボードに保存します。

## データ仕様について

本アプリは `data/cards.json` からカードデータを読み込んで動作します。
データ構造や検索ロジックの詳細については [DATA_SPEC.md](DATA_SPEC.md) をご参照ください。
